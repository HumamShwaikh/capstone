1. Open the MATLAB folder and run 'MAIN.m'
2. Use the sliders to select your desired: Diving Depth, Max Speed, and Operating Time
3. Click 'Generate' which will modify the SolidWorks equations file
4. Rebuild the FINAL_ASSEMBLY.sldasm

NOTE: This project uses relative paths, so as long as all the folders stay together. They do not need to be on a specific drive.
